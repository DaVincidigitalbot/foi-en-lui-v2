## Technical Specifications

### Design Files:
- **Main designs:** `/women/`, `/men/`, `/teens/` - 4500x5400px HTML
- **Mockups:** `/mockups/` - 1200x1500px garment mockups  
- **Listings:** `/listings/` - Etsy-ready product descriptions

### Foi en Lui Shield Logo:
```svg
Navy shield (#1a1a2e), gold border (#C9A84C)
Gold cross, letters F E L
Pointed bottom, traditional heraldic shape
Integrated into every design
```

### Embroidery CSS Technique:
```css
.embroidered {
    color: #C9A84C;
    text-shadow: 
        1px 1px 0px rgba(0,0,0,0.3),
        2px 2px 1px rgba(0,0,0,0.2),
        -1px -1px 0px rgba(255,255,255,0.1),
        0px 3px 2px rgba(0,0,0,0.15);
    background-image: repeating-linear-gradient(
        45deg, transparent, transparent 2px,
        rgba(255,255,255,0.03) 2px,
        rgba(255,255,255,0.03) 4px
    );
    -webkit-background-clip: text;
    filter: contrast(1.1);
}
```
