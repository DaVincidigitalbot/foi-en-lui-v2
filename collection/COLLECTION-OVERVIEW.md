# Foi en Lui V2 Collection - Complete Overview

## Collection Philosophy

**Foi en Lui** ("Faith in Him") represents premium faith-based apparel that refuses to compromise style for substance. This collection features **embroidered-style designs** with authentic scripture verses, targeting young Christians who want to wear their faith boldly yet sophisticatedly.

### Brand Colors
- **Navy**: #1a1a2e (Primary)
- **Gold**: #C9A84C (Accent/Thread)
- **Cream**: #F5F0E8 (Secondary)

### Design Standards
- All designs feature **embroidered texture effects** (raised, stitched appearance)
- **Scripture-only** content - no generic phrases
- **Foi en Lui shield logo** integration (shield with gold cross, F-E-L letters)
- **Premium typography** using Crimson Text serif font family
- **4500x5400px print area** optimized for Printful DTG

## PRODUCTION SPECIFICATIONS

### Printful DTG Recommendations
- **Base Garments**: 
  - Hoodies: Gildan 18500 Heavy Blend or equivalent
  - Tees: Bella+Canvas 3001 Unisex Jersey or Gildan 64000 Softstyle
- **Print Method**: Direct-to-Garment (DTG) printing
- **Print Area**: 4500x5400px designs optimized for chest placement
- **Colors**: All designs use high-contrast colors suitable for DTG printing

### File Delivery
- **Format**: HTML files for rendering to PNG via wkhtmltoimage
- **Resolution**: 300 DPI when converted to PNG
- **Background**: Transparent or matching garment color
- **Fonts**: Google Fonts integration (Crimson Text) for consistent rendering
